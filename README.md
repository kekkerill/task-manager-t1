# Task Manager

## Возможности

- Создание, редактирование и удаление задач
- Фильтрация задач по категории, статусу и приоритету
- Просмотр подробной информации о задаче
- Адаптивный дизайн для мобильных

## Стек

- React + TypeScript
- styled-components
- @admiral-ds/react-ui
- react-router-dom

## Запуск проекта

- npm install
- npm run dev
